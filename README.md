Stenographic flag generator, inspired by free speech flags related to "illegal numbers".

![](./public/screenshot.png)

## Building

Install dependencies:

```
npm install
```

Run build script:

```
npm run build
```

## Testing

Run test script:

```
npm test
```

## Running locally

Run live server script:

```
npm run dev
```
